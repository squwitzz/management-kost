# Cara Install Kost App di HP

## Android (Chrome)

1. Buka app di Chrome browser
2. Tunggu beberapa detik, akan muncul popup "Install Kost App"
3. Klik tombol **"Install"**
4. App akan muncul di home screen HP kamu

**Atau cara manual:**
1. Buka menu Chrome (3 titik di pojok kanan atas)
2. Pilih **"Add to Home screen"** atau **"Install app"**
3. Klik **"Install"**
4. Done! App sudah ada di home screen

## iPhone (Safari)

1. Buka app di Safari browser
2. Klik tombol **Share** (kotak dengan panah ke atas)
3. Scroll ke bawah, pilih **"Add to Home Screen"**
4. Klik **"Add"**
5. App akan muncul di home screen iPhone kamu

## Keuntungan Install App

✅ Akses lebih cepat (tidak perlu buka browser)
✅ Notifikasi real-time langsung ke HP
✅ Tampilan full screen seperti app native
✅ Bisa digunakan offline (untuk beberapa fitur)
✅ Icon di home screen seperti app biasa

## Enable Notifikasi

Setelah install, pastikan notifikasi aktif:

### Android
1. Buka app
2. Browser akan minta permission "Allow notifications"
3. Klik **"Allow"**

### iPhone (iOS 16.4+)
1. Buka app
2. Safari akan minta permission
3. Klik **"Allow"**

## Notifikasi yang Akan Kamu Terima

- 💰 Tagihan baru
- ✅ Pembayaran diverifikasi
- 🔧 Update maintenance request
- 📢 Pengumuman penting

## Troubleshooting

**Q: Popup install tidak muncul?**
A: Refresh halaman atau tunggu 3-5 detik

**Q: Notifikasi tidak muncul?**
A: Check settings HP → Notifications → Kost App → pastikan ON

**Q: App tidak bisa dibuka?**
A: Pastikan koneksi internet aktif

**Q: Cara uninstall?**
A: 
- Android: Long press icon → Uninstall
- iPhone: Long press icon → Remove App

## Butuh Bantuan?

Hubungi admin jika ada masalah dengan instalasi atau notifikasi.
